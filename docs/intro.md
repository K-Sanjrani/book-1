## What You'll Learn

### Chapter 1: Physical AI Fundamentals (Now Available)

**Lesson 1.1: What is Physical AI?** (~30 min)
- Define Physical AI and differentiate from traditional AI
- Explore real-world applications (smart homes, robotics, environmental monitoring)
- Understand the hardware-software interaction model
- Run your first sensor example (simulator-based)

**Lesson 1.2: Understanding Sensors** (~45 min)
- Learn common sensor types (temperature, motion, light, pressure)
- Understand analog vs. digital signals
- Read multiple sensors and log data to CSV
- Complete checkpoint: Add a new sensor type

**Lesson 1.3: Building Your First Project** (~60 min)
- Design a smart room monitoring system
- Integrate multiple sensors with decision logic
- Implement "if-then" rules for automation
- Run on real Raspberry Pi (optional)
- Complete checkpoint: Extend with humidity and comfort zones

**Total Time**: 2-3 hours + optional hardware exploration

### Future Chapters (Planned v2.0+)
- **Chapter 2**: Home Automation Project (multi-sensor coordination, logging, alerts)
- **Chapter 3**: Real-World IoT Systems (cloud integration, data analysis, deployment)

## How to Use This Book

### Path 1: Learning (Individual)
1. Read Lesson 1.1 intro
2. Run code example in simulator (no hardware needed)
3. Complete checkpoint exercise
4. Move to next lesson

**Time**: 2-3 hours | **Cost**: $0 | **Hardware**: None

### Path 2: Learning + Hardware (Individual)
1. Follow Path 1 for Lessons 1-2
2. For Lesson 1.3, set up Raspberry Pi with sensors (~$30)
3. Run examples on real hardware
4. Modify config to experiment with different thresholds

**Time**: 3-4 hours (+ hardware setup) | **Cost**: ~$30 | **Hardware**: Pi + sensors

### Path 3: Teaching (Classroom)
1. Assign Lesson 1.1 to students
2. Use provided checkpoint exercises for homework
3. Grade using provided rubric and sample solutions
4. Demonstrate Lesson 1.3 project in class (optional hardware demo)

**Time**: 2-3 class periods | **Cost**: Flexible (simulator-only is free) | **Hardware**: Optional

## Prerequisites

### Minimum (for Lessons 1.1 & 1.2)
- **Python 3.9 or higher** (download at python.org)
- **Text editor** (VS Code, PyCharm, or any editor)
- **Time**: 2-3 hours
- **Cost**: $0

### For Lesson 1.3 (Optional Hardware)
- **Raspberry Pi 4** (or Pi 3B+)
- **DHT22 temperature/humidity sensor** (~$5)
- **PIR motion sensor** (~$5)
- **Jumper wires** (~$3)
- **Breadboard** (~$2)
- **Time**: Additional 30 minutes setup + soldering (optional)
- **Cost**: ~$30 total
